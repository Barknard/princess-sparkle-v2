  - Frame 2: shift sprite 1px up (bounce effect)

## Fallback

If real sprites are unavailable, the game uses procedural placeholder sprites
generated by `game/data/placeholderSprites.js`. These draw colored shapes on
canvas elements so the game remains fully playable and visually testable.
